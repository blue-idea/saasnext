常见问题 FAQ

我可以随时取消订阅吗？

是的，您可以随时在后台设置中取消订阅。在当前计费周期结束前，您依然可以访问所有付费功能。

通过选择年付方案，我们将为您提供 20% 的折扣。这意味着您只需支付 9.6 个月的费用即可获得全年的服务。

是否提供发票？
